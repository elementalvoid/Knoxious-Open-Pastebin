Knoxious-Open-Pastebin by xanmanning

A quick to set up, rapid install, two-file pastebin!

Dependencies

+ Apache2/Lighttpd/nginx/IIS + PHP5 + PHP5-mcrypt (Optional) + MySQL (Optional)
